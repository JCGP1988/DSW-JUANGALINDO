atributos y valores en HTML5

Las etiquetas son la estructura básica del HTML. Estas etiquetas o tags se componen y contienen otras propiedades, como son los atributos y el contenido.
de estos encontramos diferentes tipos los cuales van definidos por cada etiqueta encontramos algunos comunes a muchas o casi todas las etiquetas, que se dividen en cuatro grupos según su funcionalidad:

1- Atributos básicos
2- Atributos de internacionalización
3- Atributos de eventos
4- Atributos de foco
ATRIBUTOS BASICOS
Los atributos básicos se utilizan en la mayoría de etiquetas HTML y XHTML, aunque adquieren mayor sentido cuando se utilizan hojas de estilo en cascada (CSS):
el atributo id="texto" Establece un indicador único a cada elemento
el atributo class="texto" Establece la clase CSS que se aplica a los estilos del elemento
el atributo style="texto" Aplica de forma directa los estilos CSS de un elemento
ATRIBUTO DE INTERNACIONALIZACION
Estos atributos se utilizan en aquellas páginas que muestran sus contenidos en varios idiomas y las que quieran indicar de forma explícita el idioma de sus contenidos:
el atributo lang="codigo" indica el idioma del elemento
el atributo dir indira la direccion del texto
ATRIBUTOS DE EVENTOS
Estos atributos se utilizan en las páginas web que incluyen código JavaScript para realizar acciones dinámicas sobre los elementos de la página.
- el atributo "onclick" Ejecuta la acción cuando se realiza un clic sobre el elemento
- el atributo "onmousedown" Ejecuta la acción cuando se detecta el botón pulsado del ratón
- el atributo "onmouseout" Ejecuta la acción cuando el ratón abandona el elemento
- el atributo "onmouseover" Ejecuta la acción cuando se detecta que el ratón se sitúa sobre el elemento
es importante anotar que estos tambien pueden ser distribuidos dependiendo la seccion donde se quieran ejecutar por ejemplo el body o footer o por varios.
ATRIBUTOS DE FOCO
Se le denomina foco o focus, cuando un control o elemento del documento ha sido seleccionado.
Cuando ese elemento deja de estar seleccionado, "pierde el foco" y es el nuevo elemento seleccionado el que se dice que tiene "el foco".
- el atributo accesskey="letra" Establece una tecla de acceso rápido a un elemento HTML - el atributo tabindex="numero" Establece la posición del elemento en el orden de tabulación de la página (valor entre 0 y 32.767)
